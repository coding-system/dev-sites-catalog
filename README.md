# Dev-Cat Portfolio Viewer

Веб-приложение для просмотра портфолио разработчиков из общего списка. Загружает ссылки из GitHub README, случайно переключает сайты, позволяет копировать адрес и открывать сайт в новой вкладке.

## Возможности

- Автоматическая загрузка списка портфолио из GitHub.
- Случайная навигация по сайтам, история переходов и возврат назад.
- Быстрое копирование текущей ссылки кликом по адресной строке.
- Открытие текущего сайта в новой вкладке.
- Переключение разрешения предпросмотра (FHD/2K).
- Кнопка обновления iframe.
- Горячие клавиши: стрелки влево и вправо.
- Кэширование списка ссылок на 5 минут.

## Как работает

1. Модуль [scripts/links.js](scripts/links.js) запрашивает README репозитория
   [emmabostian/developer-portfolios](https://github.com/emmabostian/developer-portfolios)
   через GitHub API.
2. Markdown парсится регулярным выражением, берутся только валидные HTTP/HTTPS ссылки.
3. В [index.js](index.js) случайно выбирается сайт и загружается в iframe.
4. Навигация управляется кнопками и клавиатурой, история переходов хранится в памяти.

## Быстрый старт

1. Скачайте репозиторий.
2. Откройте [index.html](index.html) в браузере.
3. Дождитесь загрузки списка и используйте навигацию.

## Управление и навигация

- Кнопки влево/вправо: переход по случайным сайтам.
- Стрелки клавиатуры: то же самое.
- Клик по адресу: копирование ссылки.
- Кнопка обновления: перезагрузка текущего iframe.
- Переключатель FHD/2K: меняет масштаб предпросмотра.

## Ограничения

- Некоторые сайты запрещают встраивание в iframe через
  `X-Frame-Options` или `Content-Security-Policy`.
- В таких случаях браузер может показывать страницу ошибки внутри iframe.
